网页上的内容复制为markdown

  1. copycat: 将网页内容copy成若干种其他形式, 但是作者的主页似乎挂了. ping得通, 上不去. 20180928(zz)

  2. copy as markdown 的 作者说明:

    • 把目前的分頁標題及網址複製成 Markdown
    • 按右鍵把超連結複製成 Markdown
      • 如果你是在圖片上按右鍵,則複製出來的會是連結包圖片 [![](img url)](link url)
    • 按右鍵把圖片複製成 Markdown
    • 把目前視窗的所有分頁及網址複製成 Markdown 的列表(！)
